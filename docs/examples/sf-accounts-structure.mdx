# Salesforce Accounts Example

This example demonstrates how to generate account and contact data for a Salesforce organization.

## Goal
Create data for a healthcare system with 5 business units and a total of 1000 contacts.

## Prerequisites
- Salesforce sandbox or developer account
- At least 1005 records upload capacity on [dataloader.io](https://dataloader.io).

## Preparation Steps
None

## Data Generation Files
- <a target="_blank" href="/files/salesforce-accounts/schema.yaml" download="schema.yaml">schema file</a>
- <a target="_blank" href="/files/salesforce-accounts/plan.yaml" download="plan.yaml">plan file</a>
- <a target="_blank" href="/files/salesforce-accounts/output.yaml" download="output.yaml">output control file</a>

## Notes
1. To use [dataloader.io](https://dataloader.io) to import Salesforce records, please use "CSV" as the output format.
2. When mapping contact record, link the "Account ID" field with Lookup via: Account Name.
